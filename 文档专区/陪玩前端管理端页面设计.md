```
┌─────────────────────────────────────┐
│           陪玩师申请审核              │
├─────────────────────────────────────┤
│ 申请人：张三                         │
│ 身份证：[正面图] [反面图]（可放大）    │
│ 技能描述：擅长王者荣耀...            │
│                                     │
│ 审核结果：○ 通过  ○ 拒绝             │
│ 审核备注：[________________]         │
│                                     │
│         [取消]  [确认提交]            │
└─────────────────────────────────────┘
```

| 字段名 | 标签 | 组件类型 | 校验规则 |
|--------|------|----------|----------|
| auditStatus | 审核结果 | RadioGroup（1=通过, 2=拒绝） | required |
| auditRemark | 审核备注 | Textarea | 拒绝时 required |

**提交逻辑**：调用 `updateCoachApply({ id, auditStatus, auditRemark })`，通过时后端自动创建 play_coach 记录。
