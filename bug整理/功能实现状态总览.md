## 二、仍需关注的事项（非 Bug，需产品/运营决策）

### 2.1 需填写配置后生效的功能

| 功能 | 配置位置 | 说明 |
|------|----------|------|
| 短信发送 | `config.yaml → sms` | 填写阿里云 AccessKey + 签名 + 模板，devMode 改 false |
| 微信支付 | `config.yaml → pay.wechat` | 填写 AppID/MchID/APIKey/私钥/通知URL，devMode 改 false |
| 支付宝支付 | `config.yaml → pay.alipay` | 填写 AppID/私钥/支付宝公钥/通知URL，devMode 改 false |
| 阿里云 OSS | `upload_config` 表 | storageType=2，填写 Bucket/Endpoint/AccessKey |
| 腾讯云 COS | `upload_config` 表 | storageType=3，填写 Bucket/Region/SecretId/SecretKey |
